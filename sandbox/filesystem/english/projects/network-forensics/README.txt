# Network Forensics - Exploit Kit Analysis

Year: 2025

A good analyst has to turn messy evidence into a clear story, so I took
a captured real-world intrusion and set out to reconstruct it end to
end and communicate it clearly.

The work produced a written forensic report with indicators (IOCs), an
impact assessment and a remediation timeline, built in Security Onion
and Wireshark.

Technologies: Security Onion, Wireshark, IOC analysis, Forensic reporting
