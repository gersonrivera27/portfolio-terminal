# Network Segmentation & Zero Trust Lab (pfSense)

Year: 2024

I kept reading that segmentation stops attackers from spreading, so I
built a lab to design and test network controls that actually contain
lateral movement — and to find out where they break.

I ended up with VLAN micro-segmentation and strict Zero Trust firewall
rules, validated with packet-level analysis in Wireshark.

Technologies: pfSense, VLAN, Zero Trust, Firewall rules, Wireshark, IDS/IPS
