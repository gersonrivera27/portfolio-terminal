# StackPOS - Production Full-Stack Platform

Status: In production

Working in customer support, I kept watching the ordering and billing
system fail, so I set out to design and launch a secure multi-tenant
platform from scratch and operate it myself in production.

It now runs as a live service on a VPS (Python/FastAPI, .NET 8,
PostgreSQL, RabbitMQ, Nginx) with security built in from the start:
parameterized queries and schema-as-code.

Technologies: Python, FastAPI, .NET 8, PostgreSQL, RabbitMQ, Docker, Nginx
