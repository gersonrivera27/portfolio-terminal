# Network Forensics - Exploit Kit Analysis

Año: 2025

Un buen analista tiene que convertir evidencia desordenada en una historia clara, 
así que tomé una intrusión real capturada y me propuse reconstruirla de 
principio a fin y comunicarla claramente.

El trabajo produjo un reporte forense escrito con indicadores (IOC), una 
evaluación de impacto y una línea de tiempo de remediación, construido en 
Security Onion y Wireshark.

Tecnologías: Security Onion, Wireshark, IOC analysis, Forensic reporting
