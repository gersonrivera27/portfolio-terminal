# Network Segmentation & Zero Trust Lab (pfSense)

Año: 2024

Leía constantemente que la segmentación detiene la propagación de los atacantes, 
así que construí un laboratorio para diseñar y probar controles de red que 
realmente contengan el movimiento lateral, y para encontrar dónde fallan.

Terminé con micro-segmentación de VLANs y reglas estrictas de firewall bajo 
Zero Trust, validado con análisis a nivel de paquetes en Wireshark.

Tecnologías: pfSense, VLAN, Zero Trust, Firewall rules, Wireshark, IDS/IPS
